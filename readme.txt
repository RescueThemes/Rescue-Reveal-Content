This plugin will display content within a hidden area that will reveal once hovered over. Developed for themes by Rescue Themes.

== Description ==

When activated, this plugin will add a widget called 'Rescue Reveal Content' to your Widgets area. The widget will allow you to select an icon and enter content to display on your site.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
4. Drag/Drop the widget into your sidebar and fill in the desired information.
